#路由器设置教程
说明：
1.晚上8-10点、中午12-2点高峰，网速有所下降
2.设置完毕后，以后需要使用的时候连通电源，请等3-5分钟左右再进行使用，由于刚开机后台需要打开的功能比普通路由多，启动比较慢
3.仅作为学习使用，遵守当地法规
4.遇到突然网速下降，请等待1小时，很有可能是运营商误判断当前在使用大流量下载导致限速，可适当限制网速进行规避

注意
用送的网线把盒子（蓝色口） 与 家里自己的路由器的上网口连接在一起（一般电脑网线插的那种就是）
